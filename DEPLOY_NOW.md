## 🎯 **FASTEST METHOD: Vercel (5 Minutes)**

### **Step 1: Create GitHub Account** (if you don't have one)
👉 https://github.com/signup

### **Step 2: Create New Repository**
1. Go to: https://github.com/new
2. Repository name: `vidyalabs-website`
3. Choose **Public** or **Private**
4. **DON'T** check any boxes
5. Click **"Create repository"**

### **Step 3: Push Your Code**

GitHub will show you commands. Run these in your terminal:

```powershell
# Replace YOUR_USERNAME with your actual GitHub username
git remote add origin https://github.com/YOUR_USERNAME/vidyalabs-website.git
git branch -M main
git push -u origin main
```

**Example:**
```powershell
git remote add origin https://github.com/rajeshkumar/vidyalabs-website.git
git branch -M main
git push -u origin main
```

### **Step 4: Deploy on Vercel**

1. Go to: 👉 **https://vercel.com/signup**
2. Click **"Continue with GitHub"**
3. Authorize Vercel
4. Click **"New Project"**
5. Find `vidyalabs-website` and click **"Import"**
6. Click **"Deploy"** (don't change anything)

**⏱️ Wait 2-3 minutes...**

### **🎉 DONE! Your Website is LIVE!**

Your URL will be:
```
https://vidyalabs-website.vercel.app
```

## 🌐 **Add Custom Domain (Optional)**

Want `vidyalabs.com` instead of `.vercel.app`?

### **On Vercel:**
1. Go to your project dashboard
2. Click **"Settings"** → **"Domains"**
3. Enter your domain: `vidyalabs.com`
4. Follow DNS instructions
5. Wait 24-48 hours for DNS propagation

### **Free Domain Options:**
- **Freenom**: Free `.tk`, `.ml`, `.ga` domains
- **InfinityFree**: Free subdomain
- Or buy cheap domain: **Namecheap** (~$10/year)

## 🆘 **Troubleshooting**

### **"Permission denied" when pushing to GitHub:**
```powershell
# Use GitHub Desktop app instead
# Download: https://desktop.github.com
```

### **Build fails on Vercel:**
- Check the build logs
- Usually auto-fixes on retry
- Contact Vercel support (very helpful!)

### **Can't find repository on Vercel:**
- Click "Adjust GitHub App Permissions"
- Grant access to your repository
- Refresh and try again
